# Resource Pack Template README

This is a template resource pack for the Custom Weapon Gacha plugin.

## What's Included

- **pack.mcmeta** - Resource pack metadata (already configured)
- **models/item/** - JSON model files for custom weapons
- **textures/item/** - Placeholder texture files (REPLACE THESE!)

## Custom Model Data IDs

- Astraedge: 1001
- Blood Oath Saber: 1002
- Chronoslicer: 1003

## How to Use

1. Replace the `.txt` placeholder files in `textures/item/` with actual PNG textures
2. Optionally, edit the model JSON files for 3D models
3. Package the resource pack as a ZIP file
4. Upload to GitHub Releases or hosting service
5. Configure server.properties with the resource pack URL

For detailed instructions, see **TUTORIAL-RESOURCE-PACK.md** in the root directory.

## Notes

- Keep the folder structure exactly as is
- Texture files should be 16x16 or 32x32 PNG format
- Model files follow vanilla Minecraft item model format
